Chapter 08: Transforms and Transitions

Notes:

The transform property allows to translate, rotate, scale, and/or skew any element on the page.
Translation--> move elements left, right, up, or down.
Scale(x,y) --> scales an element horizontally then vertically.
Rotate() --> rotates an element around the point of origin by a specified angle value.
Skew(x,y) --> specifies a skew along the x and y axes.
Native CSS transitions generally require less client-side processing, so they appear smoother than Javascript transition. Ideal for mobile
